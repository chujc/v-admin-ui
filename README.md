<p align="center"> VAdmin UI</p>

项目配合[VAdmin](https://github.com/chujc/v-admin)进行中后台项目开发

## 安装依赖
```
npm install
npm install --registry=https://registry.npm.taobao.org
```
## 启动服务
```
npm run dev
```
发布
## 构建测试环境
```
npm run build:stage
```
## 构建生产环境
```
npm run build:prod
```
